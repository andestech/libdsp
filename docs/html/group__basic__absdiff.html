Detailed Description

Absolute difference functions get absolute diff values from separate source vectors and write them one-by-one into a destination vector.

Andes DSP library supports distinct absolute difference functions for the following data types: F32, F16, Q31, Q15 and Q7. These functions are introduced in the subsections below.
