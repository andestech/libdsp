Detailed Description

Addition functions add two elements from separate source vectors and write the results one-by-one into a destination vector. The behavior can be defined as follows:

Andes DSP library supports distinct addition functions for single-precision floating-point, Q31, Q15, Q7 and other data types. These functions are introduced in the subsections below.
