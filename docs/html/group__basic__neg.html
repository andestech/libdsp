Detailed Description

Negate functions negate elements of a source vector and write the results one-by-one into a destination vector. The behavior can be defined as follows:

Andes DSP library supports distinct negate functions for the following data types: single-precision floating-point, Q31, Q15 and Q7. These functions are introduced in the subsections below.
