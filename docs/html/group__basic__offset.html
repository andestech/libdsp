Detailed Description

Offset functions add each element of a source vector with a constant number and write the results one-by-one into a destination vector. The behavior can be defined as follows:

Andes DSP library supports distinct offset functions for single-precision floating-point, Q31, Q15, Q7 and other data types. These functions are introduced in the subsections below.
