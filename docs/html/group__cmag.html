Detailed Description

Complex magnitude functions compute the magnitude of a complex vector. The behavior can be defined as follows:

Andes DSP library supports distinct complex magnitude functions for the following data types: single-precision floating-point, Q31 and Q15. These functions are introduced in the subsections below.
