Detailed Description

Complex matrix multiplication functions compute the multiplication of two source complex matrices and write the result into a destination complex matrix. Figure 4 shows the multiplication of complex matrix A by matrix complex B, in which both matrices have 2 rows and 2 columns

and each multiplication is treated as a complex multiplication.

Andes DSP library supports distinct complex matrix multiplication functions for the following data types: floating-point, Q31, and Q15. These functions are introduced in the subsections below.
