Detailed Description

The generic sort function sorts elements of a vector by the algorithm and sorting order specified in its instance structure. The algorithms to be chosen from to perform the generic sorting include bitonic sort, bubble sort, heap sort, insertion sort, quick sort and selection sort. Andes DSP library only supports the generic sort function for floating-point data. The function and its instance structures are introduced in the subsections below.

Example 

      With the input size as 100, sorting order as ascending and sorting algorithm as quick
      sort, the code example of generic sorting is as follows:
         #define size 100
         riscv_dsp_sort_f32_t instance;
         float32_t src[size] = {…};
         float32_t dst[size];
         riscv_dsp_sort_init_f32(&instance, RISCV_DSP_SORT_QUICK,
         RISCV_DSP_SORT_ASCENDING);
         riscv_dsp_sort_f32(&instance, src, dst, size);
