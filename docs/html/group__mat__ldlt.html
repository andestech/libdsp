Detailed Description

Matrix LDLT decomposition functions perform the LDLT decomposition of an input matrix and then write the results into destination matrices. They will return -1 if the decomposition fails. The LDLT decomposition of a matrix A is a decomposition in the form PAP^T = LDL^T where P is a destination permutation vector, L is a destination matrix and a real lower unit triangular matrix, and D is a destination diagonal matrix. Andes DSP library supports distinct matrix LDLT decomposition functions for the following data types: single- and double-precision floating-point. These functions are introduced in the subsections below.
