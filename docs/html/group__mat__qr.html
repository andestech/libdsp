Detailed Description

Matrix QR decomposition functions perform the QR decomposition of a matrix M using the Householder algorithm, M = QR, where Q is an orthogonal matrix and R is an upper triangular.

Andes DSP library supports distinct matrix QR decomposition functions for the following data types: single-, half- and double-precision floating-point. These functions are introduced in the subsections below.
