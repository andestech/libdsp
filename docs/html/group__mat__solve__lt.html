Detailed Description

Matrix solve lower triangular functions solve the lower triangle decomposition for an input matrix and write the results into a destination matrix. The lower triangle decomposition is solved in the following form. Lx = A That is, the destination matrix x can be obtained if given the input matrix A and the lower triangular matrix L. Andes DSP library supports distinct matrix solve lower triangular functions for the following data types: single-, half- and double-precision floating-point. These functions are introduced in the subsections below.
