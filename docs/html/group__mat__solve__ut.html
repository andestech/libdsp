Detailed Description

Matrix solve upper triangular functions solve the upper triangle decomposition for an input matrix and write the results into a destination matrix. The upper triangle decomposition is solved in the following form. Ux = A That is, the destination matrix x can be obtained if given the input matrix A and the upper triangular matrix U. Andes DSP library supports distinct functions for the following data types: single-, half- and double-precision floating-point. These functions are introduced in the subsections below.
